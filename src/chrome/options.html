Options

Pairing Mode (MacOS X Simulation)

Pairing mode provides MacOS X like windows management. When one of the splitted windows got focused, the other in pair will be shown together. The pair of windows always occupy the viewport without missed another one.

Remarks: It is useless in MacOS X.
  • Enable
  • Disable

Auto Maximize Mode

After window splitted, if one of the window close, the remaining one will be maximized to occupy all the space
  • Enable
  • Disable

Bookmark

With the feature of bookmark , you could relaunch your favor site in a new tab and windows will be splitted in the way you like.
  • Clear
  • Default
